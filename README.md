# 🐍 Snake Game

A classic and interactive **Snake Game** built using **HTML, CSS, and JavaScript**. The game challenges players to control a snake, collect food, increase their score, and achieve the highest score possible before time runs out or a collision occurs.

## 🎮 Game Preview

The game includes:

- 🐍 Smooth Snake Movement
- 🍎 Food Collection System
- 📈 Real-Time Score Tracking
- 🏆 High Score Tracking
- ⏱️ Real-Time Timer
- 🎨 Theme Changer
- 🔄 Restart Game Functionality
- 📱 Responsive Design

---

## ✨ Features

### 🐍 Snake Controls
- Control the snake using keyboard arrow keys.
- Smooth movement across the game board.
- Snake grows after eating food.

### 🍎 Food System
- Random food generation.
- Score increases when food is collected.
- New food appears automatically after being eaten.

### 📈 Score Tracking
- Displays the current score in real-time.
- Updates instantly after collecting food.

### 🏆 High Score System
- Tracks the highest score achieved.
- Updates automatically when a new record is set.
- Persists during gameplay sessions.

### ⏱️ Real-Time Timer
- Live countdown timer.
- Adds challenge and excitement to gameplay.
- Game ends when the timer reaches zero.

### 🎨 Theme Changer
- Switch between different visual themes.
- Enhances user experience and customization.

### 🔄 Restart Functionality
- Restart the game instantly after Game Over.
- Resets snake position, timer, and score.

### 💀 Game Over Detection
- Detects collisions and timer expiration.
- Displays Game Over screen.
- Provides restart option.

### 📱 Responsive Design
- Works smoothly on desktop and modern browsers.
- Clean and user-friendly interface.

## 🎯 How to Play

1. Click **Start the Game**.
2. Use **Arrow Keys** to move the snake:
   - ⬆️ Up Arrow
   - ⬇️ Down Arrow
   - ⬅️ Left Arrow
   - ➡️ Right Arrow
3. Eat the food to increase your score.
4. Avoid collisions and survive until the timer ends.
5. Try to beat your **High Score**.
